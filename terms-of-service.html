Data We Collect

We collect information about you in a variety of ways. For example, we collect information you provide when you register with us, fill out a form, or use our application or services. Data we collect may include

For script users:

If you are a user who is executing a script that's protected with Luarmor, this part applies to you. We collect the following data:

  • IP addresses (This is required as a part of access control. Many similar services do this.)
      ⚠️ This information is not shared to script owner. They will not be able to see your IP address unless:
      • If they implemented their own logger in the script
      • If you triggered an anti-tamper measurement

  • Timestamps (the time when you executed the script)
    • This information is delivered to script owner if they set up webhook log delivery.

  • HWID and User-Agent headers (Does not contain sensitive information)
    • HWID is basically a hash of device properties. It does not contain any sensitive information. They are only used as a part of access control since they are mostly unique per device.
    • User-Agent is specific to executor you are using (e.g Codex, Delta, Electron) so script developer can see which executor you are using.

  • Discord ID (Not always required, but it is a part of the access control)
    • This information is only collected if you got whitelisted by the discord bot. So the bot knows what to do next time you click on 'Get Script' button.
    • You can still use a script without having a discord account at all, just by using the key directly.

  • Job ID (Only some desktop executors send this information. Android executors do not send it, so it will appear as "?" in webhook logs)

  • ❌ Roblox usernames, account IDs, in-game stats, device details etc. are not collected.
    • These are not collected or needed by Luarmor. You can verify this information by monitoring what is indexed from the Lua environment.
      However, if the script developer implemented their own webhook logger in their script, we wouldn't know.

For developers:

If you own a Luarmor API key, or use luarmor dashboard, this part applies to you.

  • IP addresses, timestamps, and data you provide by adding users.
  • Email address and discord ID (for registration)
  • Any data you provide using input boxes on the website/API. (e.g script names, webhooks, user notes) (except for the script upload button)
  • ⚠️ Scripts uploaded by you are automatically deleted from our systems upon obfuscation.
    Raw code is not stored unless you use the verified script option. Read more about that [here]
